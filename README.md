# おじゃましました

## サイト概要

旅の写真を共有し、訪れた都道府県を見える化できるサイトです。

### サイトテーマ

他ユーザから投稿された各都道府県の魅力的な写真や、自身の未訪問の都道府県を見える化することで次の目的地選びにも役立てることができます。

### テーマを選んだ理由

旅好きはみな、訪れた場所に感謝することから感謝の意を込めて「おじゃましました」と名命しました。旅好きの集うSNSで、旅の写真を共有し、同じ趣向の仲間との出会いの場となればと思い、本テーマに致しました。

### ターゲットユーザ

旅と写真好きの一般ユーザ

### 主な利用シーン

- 旅の写真を共有したい時
- 訪れた都道府県を地図上で記録したい時
- 旅の目的地を検討する時
